Publishing your site

See, the cool thing about this Repository you made in the previous page is that as soon as you create this repository its contents are pushed out to the web. But if there isn't an "index.html" file in the directory then you won't see anything.

So first you need to make an index.html file. This will be your home page. Initialize a ReadMe and fill it witha description of your website. Next, goto "create new file" and then fill it with the html code for your home page. Commit the changes, wait a while, and your website should be up and running.
