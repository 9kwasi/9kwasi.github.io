Making your account and Repository

  1. Create an account at Github.

    • Don't do something fucking stupid like make your username a refference to the greatest indie rapper in the history of the world
    • "But Karna! What if we're working on a group project?! Do we use the same-" Shut the fuck up, each member of your clique should make their own account, and be added as contributors. We'll go over this later.
  2. From this account, create the repository which you want to host your website from.
    • The name of this repository is going to be "YOURUSERNAMEYOUJUSTFUCKINGMADE.github.io"
